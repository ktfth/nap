# Napps [![Build Status](https://travis-ci.org/ktfth/napps.svg?branch=master)](https://travis-ci.org/ktfth/napps)

## Description

Text searcher

## Installation

```
[sudo] npm i -g napps 
```

### Simple usage

```
echo "some sample" | napps sample
Present: sample (1)
```

```
cat sample.txt | napps sample --extract
```
